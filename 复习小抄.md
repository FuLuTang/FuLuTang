### Q8: 计算字符出现次数 (Function `occ`)
**题目**: 编写函数 `occ($tab, $car)`，返回字符 `$car` 在数组 `$tab` 中出现的次数。

```php
<?php
function occ($tab, $car) {
    $compteur = 0;
    foreach ($tab as $ele) {
        if ($ele == $car) {
            $compteur++;
        }
    }
    return $compteur;
}
```

### Q9: 计算最大振幅 (Function `plusGrandeAmplitude`)
**题目**: 编写函数 `plusGrandeAmplitude($tableau)`，返回数组中任意两个**相邻**元素之间差值的绝对值的最大值。

```php
<?php
function plusGrandeAmplitude($tableau) {
    $tab = $tableau;
    $ampMax = 0;
    for ($i = 1; $i < count($tab); $i++) {
        $amp = $tab[$i] - $tab[$i - 1]; // 当前元素减去前一个
        if ($amp < 0) {
            $amp = -$amp; // 取绝对值
        }
        if ($amp > $ampMax) {
            $ampMax = $amp;
        }
    }
    return $ampMax;
}
```

### Q11: 判断递减数组 (Function `estDecroissant`)
**题目**: 编写函数 `estDecroissant($tab)`，判断数组是否为严格递减序列。空数组或单元素数组返回 true。

```php
<?php
function estDecroissant($tab) {
    $size = 0;
    foreach ($tab as $nb) {
        $size++;
    }
    if ($size == 0 || $size == 1) {
        return true;
    }
    $last = 9999;
    foreach ($tab as $nb) {
        if ($nb >= $last) {
            return false;
        }
        $last = $nb;
    }
    return true;
}
```

### Q12: 统计摘要 (Function `resumeStatistique`)
**题目**: 编写函数 `resumeStatistique($tab)`，返回包含 min, max, moy (平均值), amp (幅度) 的关联数组。需先实现辅助函数 `moy($tab)`。

```php
<?php
function moy($tab) {
    $count = 0;
    $somme = 0;
    foreach ($tab as $nb) {
        $somme += $nb;
        $count++;
    }
    $res = $somme / $count;
    return $res;
}

function resumeStatistique($tab) {
    $max = max($tab);
    $min = min($tab);
    return array(
        "max" => $max,
        "min" => $min,
        "moy" => moy($tab),
        "amp" => $max - $min
    );
}
```

### Q13: 商品类 (Class `Article`)

```php
<?php
class Article {

    private string $nom;

    public static function ajouterArticleAuStock($article, $quantite) {

        if (!isset(self::$stock[$numA])) {
            self::$stock[$numA] = 0;
        }
        self::$stock[$numA] += $quantite;
    }

    public static function afficherStock($numArticle) {

            echo "Article inconnu";
        } else {
```

### Q14: 点类 (Class `Point`)
**题目**: 实现 2D 点类。包含静态方法 `distance` 和实例方法 `modify`, `decaler`。

```php
<?php
class Point {
    private string $nom;
    private float $x;
    private float $y;

    public static function distance($A, $B) {
        return sqrt(($A->x - $B->x) ** 2 + ($A->y - $B->y) ** 2);
    }

    public function __construct($im, $km, $co) {
        $this->nom = $im;
        $this->x = $km;
        $this->y = $co;
    }

    public function modifier($x, $y) {
        $this->x = $x;
        $this->y = $y;
    }

    public function decaler($dx, $dy) {
        $this->x += $dx;
        $this->y += $dy;
    }

    public function dist($A) {
        return Point::distance($this, $A);
    }
}
```

### Q15: 三角形类 (Class `Triangle`)
**题目**: 包含三角形三边长，实现几何判断方法：是否能构成三角形 (`est_realisable`)、扁平 (`est_plat`)、等边 (`est_equilateral`)、等腰 (`est_isocele`)、直角 [...]

```php
<?php
class Triangle {
    private float $cote1;
    private float $cote2;
    private float $cote3;

    public function __construct($c1, $c2, $c3) {
        $this->cote1 = $c1;
        $this->cote2 = $c2;
        $this->cote3 = $c3;
    }

    public function __toString() {
        return "(" . $this->cote1 . ";" . $this->cote2 . ";" . $this->cote3 . ")";
    }

    // 两边之和必须大于第三边 (<: 不可行)
    public function est_realisable() {
        if (($this->cote1 + $this->cote2) < $this->cote3) return false;
        if (($this->cote1 + $this->cote3) < $this->cote2) return false;
        if (($this->cote3 + $this->cote2) < $this->cote1) return false;
        return true;
    }

    // 两边之和等于第三边（退化/扁平）
    public function est_plat() {
        return ($this->cote1 + $this->cote2 == $this->cote3) ||
               ($this->cote1 + $this->cote3 == $this->cote2) ||
               ($this->cote2 + $this->cote3 == $this->cote1);
    }

    public function est_equilateral() {
        return $this->cote1 == $this->cote2 && $this->cote2 == $this->cote3;
    }

    public function est_isocele() {
        return !$this->est_equilateral() &&
            ($this->cote1 == $this->cote2 || $this->cote1 == $this->cote3 || $this->cote2 == $this->cote3);
    }

    // 勾股定理 a^2 + b^2 = c^2
    public function est_rectangle() {
        $c1 = $this->cote1;
        $c2 = $this->cote2;
        $c3 = $this->cote3;
        if (($c1 * $c1 + $c2 * $c2) == $c3 * $c3) return true;
        if (($c1 * $c1 + $c3 * $c3) == $c2 * $c2) return true;
        if (($c3 * $c3 + $c2 * $c2) == $c1 * $c1) return true;
        return false;
    }

    public function est_quelconque() {
        return $this->est_realisable() && !$this->est_plat() &&
        !$this->est_equilateral() && !$this->est_isocele() && !$this->est_rectangle();
    }
}
```
